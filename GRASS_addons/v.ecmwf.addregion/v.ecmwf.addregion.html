DESCRIPTION

v.ecmwf.addregion adds a new partner region to existing partner regions. Geometries are automatically cleaned and combined. The combination requires a column to be used for merging (default: DISS_CENTR). This column holds the code for thw dissemination center that takes responsibility for the new region.

Topology cleaning

When importing polygons, non-topological polygons are converted to topological areas. Most potential errors in the input polygons (unexpected overlapping areas, small gaps between polygons, or warnings about being unable to calculate centroids) are automatically cleaned.

EXAMPLE

# import SHAPE file at full extent and reproject to current location projection
v.ecmwf.addregion partner_regions=efas_partner_regions_epsg3857.shp \
                  new_partner=galicia_region.shp \
		  output=efas_partner_regions_epsg3857_v2.shp

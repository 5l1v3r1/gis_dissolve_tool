DESCRIPTION

v.ecmwf.addregion adds a new partner region to existing partner regions. Geometries are automatically cleaned and combined. The combination requires a column to be used for merging (default: DISS_CENTR). This column holds the code for the dissemination center that takes responsibility for the new region.

Topology cleaning

When importing polygons, non-topological polygons are converted to topological areas. Most potential errors in the input polygons (unexpected overlapping areas, small gaps between polygons, or warnings about being unable to calculate centroids) are automatically cleaned.

Gaps smaller than mingapsize are automatically removed. If too many gaps are still left, try with a larger mingapsize. The default is 10000000, recommended larger values are the current value increased by 10000000, up to a mingapsize of 50000000.

EXAMPLE

Adding Galicia to the existing partner regions. 
v.ecmwf.addregion partner_regions=efas_partner_regions_epsg3857.shp \
                  new_partner=galicia_region.shp \
		  output=efas_partner_regions_epsg3857_v2.shp
